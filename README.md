# Map Maze Using SFML

A utility to visualize search algorithms such as A* and Dijkstra

## Steps taken:
- create src, assets, dependencies dirs
- DL SFML and ad lib include folders to dependencies
- set up CMake, CMakeLists.txt

```bash
mkdir build
cd build
cmake ..
make
```
